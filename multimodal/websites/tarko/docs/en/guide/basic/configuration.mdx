# Configuration

Tarko Agents are configured through a comprehensive configuration object that defines the Agent's behavior, capabilities, and runtime settings.

## Basic Configuration

```typescript
import { Agent } from '@tarko/agent';

const agent = new Agent({
  // Agent identity
  name: 'MyAgent',
  description: 'A helpful assistant',
  
  // Model configuration
  model: {
    provider: 'openai',
    apiKey: process.env.OPENAI_API_KEY,
    model: 'gpt-4'
  },
  
  // System prompt
  systemPrompt: 'You are a helpful assistant...',
  
  // Tools
  tools: []
});
```

## Configuration Options

### Agent Identity

```typescript
{
  name: 'string',           // Agent name
  description: 'string',    // Agent description
  version: 'string'         // Agent version
}
```

### Model Configuration

```typescript
{
  model: {
    provider: 'openai' | 'anthropic' | 'volcengine' | 'custom',
    apiKey: 'string',
    model: 'string',
    baseURL?: 'string',     // For custom providers
    temperature?: number,   // 0-1, default: 0.7
    maxTokens?: number,     // Max response tokens
    topP?: number          // Nucleus sampling
  }
}
```

### Context Engineering

```typescript
{
  context: {
    maxLength: 32000,       // Max context length
    compressionRatio: 0.8,  // Context compression ratio
    retentionStrategy: 'recent' | 'important' | 'hybrid'
  }
}
```

### Tool Call Engine

```typescript
{
  toolCallEngine: {
    type: 'native' | 'prompt-engineering' | 'kor',
    maxRetries: 3,
    timeout: 30000
  }
}
```

### Agent Hooks

```typescript
{
  hooks: {
    beforeToolCall?: (toolCall) => void,
    afterToolCall?: (result) => void,
    onError?: (error) => void,
    onComplete?: (result) => void
  }
}
```

## Environment Variables

Common environment variables for **Tarko Agents**:

```bash
# Model Provider API Keys
OPENAI_API_KEY=your_openai_key
ANTHROPIC_API_KEY=your_anthropic_key
VOLCENGINE_API_KEY=your_volcengine_key  # For Doubao models
ARK_API_KEY=your_ark_key               # Alternative for Volcengine

# Optional: Custom endpoints
OPENAI_BASE_URL=https://your-proxy.com/v1
ANTHROPIC_BASE_URL=https://your-proxy.com

# Development settings
NODE_ENV=development
LOG_LEVEL=debug
```

### Example .env file

```bash
# .env
OPENAI_API_KEY=sk-...
VOLCENGINE_API_KEY=your-volcengine-key
LOG_LEVEL=info
```

## Configuration File

You can also use a configuration file (`tarko.config.js`):

```javascript
module.exports = {
  name: 'MyAgent',
  description: 'A helpful assistant',
  model: {
    provider: 'openai',
    apiKey: process.env.OPENAI_API_KEY,
    model: 'gpt-4'
  },
  tools: [
    '@tarko/tools/file-system',
    '@tarko/tools/browser'
  ]
};
```

## Validation

Tarko validates configuration at startup and provides helpful error messages:

```typescript
try {
  const agent = new Agent(config);
} catch (error) {
  console.error('Configuration error:', error.message);
}
```
