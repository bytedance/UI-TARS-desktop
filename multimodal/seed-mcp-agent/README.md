## 配置模型 和 mcp apiKey

```
# bernald proxy
export SEED_PROXY_URL=""
# seed 模型 apikey
export ARK_TEST_KEY=""
# tavily api key
export TAVILY_API_KEY=""
# google search mcp url
export GOOGLE_MCP_URL=""
# google search api key
export GOOGLE_API_KEY=""
```

pnpm dev:agent
